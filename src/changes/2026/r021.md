# Release 2026_021 (2026-06-01)

## Impact

### 26.05

None, this is an initial release of a new platform version. No existing customer machines are upgraded automatically.


## NixOS 26.05 platform

This is the initial production release of the 26.05 platform. As of June 1, 2026 it became the default release for new virtual machines. The 25.11 platform is still maintained until June 30, 2026. To continue receiving security updates, please update to 26.05 soon.

See [Platform Upgrades & What’s New](https://doc.flyingcircus.io/roles/fc-26.05-production/upgrade.html) for things to consider before upgrading, breaking, and significant changes and new package versions.
